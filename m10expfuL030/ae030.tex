	{\bf Exponential function and exponential equations}
	
	\begin{enumerate}[1.]
		{\bf \item Given is the general exponential function $ f(x) = b \cdot a^x $}
		\begin{enumerate}[a)]
			\item Name and define the parameters $ a, b $ and specify the domain.
			\item Justify your choice of the number range for the base using three arguments for the ranges $ a < 0 ; \quad a = 0 \quad $ and $ \quad a = 1 $ (possibly calculation example).
			\item Provide an example function for each and explain your choice:
			
			(i): The graph is monotonically decreasing and does not pass through the point $ P(0|1) $
			
			(ii): The graph is monotonically increasing and passes through the point $ Q(0|2) $
		\end{enumerate}
		
		{\bf \item Find all solutions to the following exponential equations.}
		
		\begin{enumerate}[a)]
			\item $ 4^{2x-5}=16 $
			\item $ \frac{1}{3} \cdot 3^{x+1} + 3 = $30
			\item $ 2^{x^2-x-7}=\frac{1}{32} $
		\end{enumerate}
		
		{\bf \item The points $ P(2|36) $ and $ Q(5|\frac{32}{3})$ lie on a general exponential function $ f(x) = b \cdot a^x $. }\\
		Determine the values of $b$ and $a$ and give the appropriate function rule.
