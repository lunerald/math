\begin{enumerate}
\item Cut out all the cards and shuffle them
\item Which functions are not reversible?
\item Which functions are identical again after the reversal?
\item Assign the invertible function graphs in pairs to their inverse functions.
\item For each function, select the function term from the cards and determine the respective definition and value set.
\end{enumerate}
