		{\bf \item Given are the points $O(0|0)$, $A(2|0)$, $B(6|-1)$. A square should be constructed mathematically.}
		
		\begin{enumerate}[a)]
			
			\item Calculate the vector $\overrightarrow{AB}$
			\item Construct a vector that is orthogonal to $\overrightarrow{AB}$ and has the same length
			\item Use this to calculate a possible point $C$
			\item Add a point $D$ so that square ABCD becomes complete.
			\item Draw the square in the diagram and compare your results.
			
			\begin{minipage}{0.4\textwidth}
				\item Write a function term for the line $AB$ in parameter form
				\item Convert the parameter form to normal form
				\item What angle does the line $AB$ make with the x-axis?
				\item Create an auxiliary line perpendicular to $AB$ through the origin.
				\item Determine the plumb point L of the origin on the line $AB$
				\item Determine the distance $|\overrightarrow{OL}|$of the origin from the line $AB$
			\end{minipage}
