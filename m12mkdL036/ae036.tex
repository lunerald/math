	{\bf pattern curve discussion}
	
	\begin{enumerate}[1.]
		{\bf \item The function $$f(x)=-x^3+3x^2$$ is given.
			Conduct a full curve discussion. }
		\begin{enumerate}[a)]
			\item Find the definition set, the behavior at infinity and the value set.
			\item Examine the function for symmetry.
			\item Calculate all zeros and their multiples.
			\item Take the first, second and third derivatives.
			\item Investigate the monotonicity behavior and the type and location of the extreme points using a monotonicity table.
			\item Determine the curvature behavior and the turning point.
			\item Specify the term of the turning tangent.
			\item Draw the graph $G_f$ with the turning tangent in a suitable coordinate system based on the knowledge gained.
			\item The graph $G_f$ and the $x$ axis enclose a patch. Calculate its contents.
		\end{enumerate}
		
	\end{enumerate}
