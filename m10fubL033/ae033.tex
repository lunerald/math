			\item Show that $ x=3 $ is a zero of the function $ g $.

			\item Find the coordinates of the intersection of $ g $ with the y-axis.
			\item Determine the behavior of the graph of $ g $ for $ x \longrightarrow \infty $
			and $ x \longrightarrow -\infty $ . Give a brief explanation for your result.
		\end{enumerate}

		%Exercise 2
		{\bf \item The function is given }
		$$ f(x) = \frac{2x^3-2x^2-8x+8}{x^2+2x-3} $$
		\begin{enumerate}[a)]
			\item Use a calculation to show that: $ D_f = \mathbb{R} \setminus \{-3;1\}$
			\item Represent the function term in fully factored form.
			\item Use comprehensible limit value considerations to justify that the function $f$ has a defeasible definition gap and calculate all zeros of the function $f$.
			\item Justify by comprehensible considerations that the function $f$ has a pole and specify the position and type of the pole.
			\item Determine the type and equations of all asymptotes of the function.
		\end{enumerate}

		{\bf \item Given is the function $f$ with $f(x)= \frac{1}{x+2}+5$.}

			\item Determine the largest possible definition set $D_f$ and the equations of all horizontal and vertical asymptotes of the function graph $G_f$.
			\item Find the x-location where the function value is 7.
			\item The graph $G_g$ of the function $g$ is created from $G_f$ by stretching in the x-direction by a factor of 0.5 and then shifting by +1 in the y-direction. Calculate the function term $g(x)$ and the equations of the horizontal and vertical asymptotes of $G_g$
		\end{enumerate}
	\end{enumerate}
