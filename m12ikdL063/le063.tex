	\noindent{\bf Task 1. a)}\\
	The function we are looking for should be of the fourth degree, so our basic form is:
	$$f(x)=ax^4+bx^3+cx^2+dx+e$$
	A completely rational function is axisymmetric if it only has even powers of x. This reduces our form to:
	$$f(x)=ax^4+cx^2+e$$
	Now we insert the zero and get:
	$$f(0)= 0 \quad \Rightarrow \quad e=0$$
	Next we set the high point:
	$$f(1)=2 \quad \Rightarrow \quad a + c = 1 \quad I$$
	At the high point the derivative is zero:
	$$f'(x)=4ax^3 + 2cx \stackrel{!}{=}0 \quad \Rightarrow \quad f'(1)= 4a + 2c =0 \quad II$$
	We solve I for a and substitute in II:
	$$4(1-c) + 2c =0 \quad \Rightarrow \quad c= c=2 $$
	This results in $a=-1$
	So the function term is:
	$$f(x)=-x^4+2x^2$$
	
	\noindent{\bf Task 1. b)}\\
	First we need the zeros on the right and left:
	$$f(x)=-x^4+2x^2=0 \quad \Rightarrow \quad x_{1/2}=\pm\sqrt{2}$$
	The area is equal to the integral between:
	$$\int\limits_{-\sqrt{2}}^{\sqrt{2}}(-x^4+2x^2) {\rm d} x =\left[-{1\over 5}x ^5+{2 \over 3}x^3\right]_{-\sqrt{2}}^{\sqrt{2}} = {16\over 15}\sqrt{2} \approx 1.51~ {\rm FE} $$
	
	\noindent{\bf Task 1. c)}\\
	We subtract the function from the line $y=1$ and integrate:
	$$\int\limits_{-1}^1 (1-(-x^4+2x^2)){\rm d} x =\left[x+{1\over 5}x^5-{2 \ over 3}x^3\right]_{-1}^1={16 \over 15}~{\rm FE} $$

	\noindent {\bf Task 5. a)}\\
	The parallel to the x-axis has the functional equation $y=k^2$.
	The value of the integral as an area balance should then become zero:
	$$\int\limits_0^1 (x^2 - k^2) {\rm d} x =0$$
	This means that for $k$:
	$$
	\renewcommand{\arraystretch}{2}
	\begin{array}{rcll}
		\left[{1\over3 }x^3 - k^2 x \right]_0^1 &=& 0 \\
		{ 1\over 3} - k^2 &=& 0 \\
		k &=& \pm \sqrt{1 \over 3} \\
	\end{array}
	$$
	Only the positive solution is possible, so $k={1\over 3} \sqrt{3}$.

	\noindent{\bf Task 8. a)}\\
	We take the first derivative and set it equal to zero:
	$$
	\renewcommand{\arraystretch}{2}
	\begin{array}{rcll}
		f'(x)&=&{3\over 4}x^2-6x+9=0 \\
		0&=& x^2 -8x +12 \\
		x_1&=& 6 \\
		x_2&=& 2
	\end{array}
	$$
	The second derivative is:
	$$
	\renewcommand{\arraystretch}{2}
	\begin{array}{rccll}
		f''(x) &=& {3\over 2}x-6 \\
		f''(x_1) &=& 9 -6 = 3&> 0 & \Rightarrow T (6|f(6))\\
		f''(x_2) &=& 3 -6 = -3&< 0 & \Rightarrow H (2|f(2))\\
	\end{array}
	$$
	The extreme points are: $T(6|0)$ and $H(2|8)$.
	
	\noindent{\bf Task 8 . b)}\\
	For the turning point we set the second derivative equal to zero and get:
	$$f''(x) = {3\over 2}x-6 = 0 \quad \Rightarrow x_w =4 $$
	We test the third derivative at this point:
	$$f'''(4)={3 \over 2} \not = 0$$
	So we have an inflection point at $W(4|f(4))=(4|4)$
