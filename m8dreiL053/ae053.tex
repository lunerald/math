		{\bf \item Judge the following statements as true or false!} Give a counterexample for the false statements.
		
		\begin{enumerate}[a)]
			\item For all point-symmetric quadrilaterals, adjacent angles add up to $180^\circ$.

			\item If the opposite sides of a quadrilateral are the same length, then it is axially symmetrical.
			\item If the diagonals of a quadrilateral are the same length, then it is point-symmetrical.
		\end{enumerate}
