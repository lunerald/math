	\noindent{\bf Task 1. a)}\\
	The relative frequency is:
	$$h(3)={8\over 40} = { 1 \over 5}$$
	
	\noindent{\bf Task 1. b)}\\
	$$H(4)= 40 \cdot 0.125 = 5 $$
	The number 4 was thrown 5 times.
	
	\noindent{\bf Task 1. c)}\\
	The absolute frequency of 6 is:
	$$H(6)=40-(7+4+8+5+6)=10$$
	
	\noindent{\bf Task 1. d)}\\
	We add the frequencies:
	$$H(X\le 3)=H(1)+H(2)+H(3)= 7+4+8=19$$

	\noindent{\bf Task 4.}\\
	The result space is:
	$$\Omega=\{WW; MM; MW; WM\}$$
	All elements have the same probability $p={1 \over 4}$. The event set E= "at least one boy" consists of three elements, so the probability of this is:$p(E)={3 \over 4}$
	
	\noindent{\bf Task 5.}\\
	The event set is:
	$$E=\{55; 56; 65; 66\}$$
	Each element has probability $p={1 \over 36}$, so is
	$$p(E)={4 \over 36}={1 \over 9}$$
